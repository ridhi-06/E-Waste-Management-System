# E-Waste Management System

A robust backend Java application designed to address the challenges of electronic waste lifecycle management by streamlining sorting, categorization, and routing workflows.

# Core Technical Features
- **Object-Oriented Architecture:** Implemented strong OOP principles (Inheritance, Encapsulation, Polymorphism) to handle diverse electronic asset types (e.g., consumer devices, industrial hardware).
- **Automated Routing Logic:** Developed a backend processing pipeline that categorizes waste based on condition and component value, matching them to certified regional recycling centers.
- **Dynamic Inventory Analytics:** Built out systemic logic to track and display current operational processing metrics.

# Tech Stack
- **Language:** Java (JDK 11+)
- **Concepts:** Object-Oriented Programming, Data Collections, Exception Handling
